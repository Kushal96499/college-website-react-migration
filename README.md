## 🚀 Project Overview

This repository contains the React.js migration of the **Admissions page** for Biyani Group of Colleges, completed during my 1-month Web Cell internship in the 6th semester. The goal was to deliver a modern, responsive, and maintainable admissions interface, improving both user experience and code quality.

**🔗 [Live Website](https://biyaniiti.com/bgc/kushal/)**

---

## 🏆 Key Highlights

- Migrated the legacy admissions page to a dynamic React.js application using Vite.
- Designed modular, reusable components for scalability and maintainability.
- Enhanced performance, responsiveness, and accessibility.
- Collaborated with the Web Cell team for seamless integration and deployment.
- Followed best practices in code quality, version control, and UI/UX.

## 📝 How to Run

1. Clone the repository:
   ```sh
   git clone https://github.com/Kushal96499/college-website-react-migration.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
